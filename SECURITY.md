# Security Policy

## Supported Versions

We support the latest version of AegisGraph Sentinel. Please update to the latest release to receive security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly by contacting the maintainers directly instead of creating a public issue.

You can report vulnerabilities by opening a draft security advisory in this repository. We will investigate the issue and coordinate a fix.
